## 1. 前言

### 1.1 系统简介

这是一个运行在 **Polymarket 预测市场**上的自动化交易机器人。它交易"BTC 15 分钟涨跌"市场，每 15 分钟一个交易周期，使用 6 种信号处理器 + 趋势过滤器来判断方向。

### 1.2 核心原则

- **每笔固定 $1.00**，风险严格可控
- **模拟模式优先**，默认始终为模拟交易
- **无需 24/7 运行**，bot 有内置 90 分钟自动重启机制
- **先模拟后实盘**，建议至少模拟 100+ 笔交易后再考虑实盘

### 1.3 你需要准备的

| 项目 | 要求 | 费用 |
|------|------|------|
| Python | 3.14+ | 免费 |
| Redis | 任意版本 | 免费 |
| Polymarket 账号 | 需创建 API Key | 免费 |
| Polygon 钱包 | 有 USDC 余额（实盘需要） | 少量 gas 费 |
| Grafana（可选） | 可视化监控 | 免费 |

## 2. 环境准备

### 2.1 安装 Python 3.14+

**下载地址**: https://www.python.org/downloads/

安装时务必勾选 **"Add Python to PATH"**。

验证安装：

```bash
python --version
# 输出: Python 3.14.x 或更高
```

### 2.2 安装 Redis

Redis 用于在不重启 bot 的情况下切换模拟/实盘模式。

**Windows**:
```
下载 https://github.com/microsoftarchive/redis/releases
安装 Redis-x64-xxx.msi
启动: redis-server
```

**macOS**:
```bash
brew install redis
brew services start redis
```

**Linux**:
```bash
sudo apt install redis-server
sudo systemctl start redis
```

验证 Redis 是否运行：
```bash
redis-cli ping
# 输出: PONG
```

### 2.3 获取 Polymarket API 凭证

1. 用你的 **Polygon 钱包**（MetaMask 等）登录 https://polymarket.com
2. 进入 **Settings → API**
3. 点击 **Create API Key**
4. 记录以下三项：
   - **API Key** → `POLYMARKET_API_KEY`
   - **Secret** → `POLYMARKET_API_SECRET`
   - **Passphrase** → `POLYMARKET_PASSPHRASE`
5. 同时准备好钱包私钥和钱包地址

> ⚠️ 你的钱包地址可以从 MetaMask 或通过以下方式获取：
> ```bash
> python -c "from eth_account import Account; print(Account.from_key('0x你的私钥').address)"
> ```

## 3. 安装与配置

### 3.1 下载项目

```bash
git clone https://github.com/yourusername/polymarket-btc-15m-bot.git
cd polymarket-btc-15m-bot
```

### 3.2 创建虚拟环境

```bash
# Windows
python -m venv venv
venv\Scripts\activate

# macOS / Linux
python -m venv venv
source venv/bin/activate
```

激活后命令行前面会出现 `(venv)` 标识。

### 3.3 安装依赖

```bash
pip install -r requirements.txt
```

安装时间取决于网络状况，通常 2-5 分钟。如果安装慢，可以换国内镜像：

```bash
pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple
```

### 3.4 配置环境变量

复制配置模板：

```bash
cp .env.example .env
```

编辑 `.env` 文件，填入你的凭证：

```ini
# ── Polymarket API Credentials ──────────────────────────
POLYMARKET_PK=abc123def456...           # 你的钱包私钥（不带 0x 前缀）
POLYMARKET_API_KEY=019cd80c-xxxx-xxxx-xxxx-xxxxxxxxxxxx  # Polymarket API Key
POLYMARKET_API_SECRET=xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx    # API Secret
POLYMARKET_PASSPHRASE=your_passphrase                     # API Passphrase
POLYMARKET_FUNDER=0x86d472d5...         # 你的 Polygon 钱包地址

# ── Redis Configuration ────────────────────────────────
REDIS_HOST=localhost
REDIS_PORT=6379
REDIS_DB=2

# ── Trading Parameters ─────────────────────────────────
MAX_POSITION_SIZE=1.0
STOP_LOSS_PCT=0.30
TAKE_PROFIT_PCT=0.20
SPIKE_THRESHOLD=0.15
DIVERGENCE_THRESHOLD=0.05
```

> ⚠️ **私钥安全**：`POLYMARKET_PK` 是你的钱包私钥，**不要分享给任何人**，**不要提交到 GitHub**（已通过 `.gitignore` 排除 `.env` 文件）。

### 3.5 验证环境配置

```bash
python -c "from dotenv import load_dotenv; load_dotenv(); print('✓ dotenv loaded')"
python -c "import redis; r = redis.Redis(); r.ping(); print('✓ Redis connected')"
```

## 5. 运行模式详解

### 5.1 三种模式对比

| 模式 | 命令 | 交易间隔 | 是否真金白银 | 用途 |
|------|------|----------|-------------|------|
| **模拟模式**（默认） | `python bot.py` | 15 分钟 | ❌ 不花真钱 | 日常运行、验证策略 |
| **测试模式** | `python bot.py --test-mode` | 1 分钟 | ❌ 不花真钱 | 快速调试、新参数测试 |
| **实盘模式** | `python bot.py --live` | 15 分钟 | ✅ 真金白银 | 真实交易 |

### 5.2 模拟模式详解

模拟模式下：

- 从 Polymarket 获取实时报价数据
- **不下真实订单**，只记录 paper trade
- 通过 `_record_paper_trade()` 模拟成交
- 数据保存在 `data/trades.db`（SQLite）+ `paper_trades.json`
- 每笔交易的 PnL 根据随机方向模拟（正向概率 ~70%）

### 5.3 测试模式详解

测试模式将 15 分钟窗口缩短到 1 分钟：

- 每次交易窗口只持续 60 秒
- 模拟退出的时间也缩短到 1 分钟
- 大幅加快验证速度

### 5.4 实盘模式

```bash
python bot.py --live
```

实盘模式下：

- 通过 Nautilus 框架连接 Polymarket CLOB
- 真实下单到 Polymarket 预测市场
- **每一笔都是真金白银**
- 下单前经过严格风控检查

> ⚠️ **首次运行务必先模拟 !** 至少运行 50-100 笔模拟交易确认策略表现后再切实盘。

## 8. 切换交易模式

### 8.1 通过 Redis 切换（无需重启）

前提：Redis 必须正在运行。

```bash
# 查看当前模式
python redis_control.py status

# 切换到模拟模式
python redis_control.py sim

# 切换到实盘模式（需要二次确认）
python redis_control.py live
```

交互式模式（直接运行）：
```bash
python redis_control.py
```

### 8.2 切换原理

```
redis_control.py set sim  →  Redis SET btc_trading:simulation_mode = 1
                                    ↓
bot.py 每 10 秒检查 Redis  →  检测到模式变化 → 切换交易逻辑
```

## 10. 常见问题排查

### 10.1 Redis 连接失败

```
Redis connection failed: Error 10061 connecting to localhost:6379
```

**解决方法**：
```bash
# 确保 Redis 正在运行
redis-server

# 测试连接
redis-cli ping
# 应该输出: PONG
```

### 10.2 API 认证失败（401）

```
PolyApiException[status_code=401, error_message={'error': 'Unauthorized/Invalid api key'}]
```

**检查清单**：

| 检查项 | 操作 |
|--------|------|
| API Key 是否正确？ | 去 Polymarket Settings → API 重新生成 |
| `signature_type` 是 0 还是 1？ | 标准 EOA 钱包用 `0`，Email 钱包用 `1` |
| 钱包是否匹配？ | API Key 必须在你当前钱包下创建 |
| API 凭证是否过期？ | 删除旧的，重新创建 |

验证钱包地址：
```bash
python -c "
from eth_account import Account
pk = open('.env').read().split('POLYMARKET_PK=')[1].split('\n')[0].strip()
print('钱包地址:', Account.from_key(pk).address)
"
```

### 10.3 POLYMARKET_FUNDER 缺失

```
RuntimeError: Environment variable 'POLYMARKET_FUNDER' not set
```

**解决方法**：在 `.env` 中添加你的 Polygon 钱包地址（就是私钥对应的公钥地址）。

### 10.4 模拟模式仍然报 401

如果只是模拟交易，不需要真实下单，bot 已经配置为**模拟模式下不创建 exec client**（不连接 Polymarket 的认证接口）。如果还报错，检查 `.env` 中的 API Key 格式是否正确（没有多余空格或引号）。

### 10.5 没有报价数据

```
No current market found — waiting for nearest future
```

**原因**：当前没有活跃的 BTC 15 分钟市场（可能是非交易时间，或 slug 格式不匹配）。

**检查**：
1. 确认 Polymarket 上确实有 BTC 15 分钟市场在交易
2. 确认 slug 格式为 `btc-updown-15m-{timestamp}`

### 10.6 Bot 自动重启

bot 内置 **90 分钟自动重启**机制：

```
AUTO-RESTART TIME - Loading fresh filters
```

这是正常行为。重启会重新加载市场列表，防止长时间运行导致的内存泄漏或市场过期。`15m_bot_runner.py` 会自动处理重启。

### 10.7 依赖安装失败

```
pip install -r requirements.txt 失败
```

尝试：
```bash
# 升级 pip
pip install --upgrade pip

# 使用国内镜像
pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple

# 分步安装
pip install nautilus_trader
pip install redis
pip install python-dotenv
pip install loguru
pip install prometheus_client
pip install eth-account
pip install web3
```
